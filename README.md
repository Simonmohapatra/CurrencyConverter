#  Currency Converter

A responsive web-based Currency Converter that allows users to convert currencies in real-time using live exchange rates. The application dynamically updates country flags based on the selected currencies and provides accurate conversion results through an external currency API.

##  Features

* Real-time currency conversion
* Supports multiple international currencies
* Dynamic country flag updates
* User-friendly and responsive interface
* Input validation for amount fields
* Fetches live exchange rates using Currency API

##  Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Font Awesome Icons
* Currency API
* Flags API

##  How It Works

1. User enters the amount to be converted.
2. Selects the source currency.
3. Selects the target currency.
4. Clicks the **Convert** button.
5. The application fetches the latest exchange rate from the Currency API.
6. The converted amount is displayed instantly.


##  Installation & Usage

1. Clone the repository:

```bash
git clone https://github.com/Simonmohapatra/CurrencyConverter.git
```

2. Navigate to the project directory:

```bash
cd CurrencyConverter
```

3. Open `index.html` in your browser.

No additional dependencies are required.

##  APIs Used

### Currency Exchange Rates

Provides real-time currency conversion data.

### Flags API

Used for displaying country flags corresponding to selected currencies.
